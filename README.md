# GridMind AI × PRIME Digital Ecosystem

An enterprise-grade smart grid orchestration, edge-intelligent energy optimization, and decentralized peer-to-peer settlement ecosystem. This repository unites the edge telemetry capabilities of **GridMind AI** with the operational management, resource planning, and financial settlement infrastructure of **PRIME Digital**.
